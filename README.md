### Preview

Program A/B Testing Portfolio Mentorship ini berfokus pada pembelajaran dan praktik langsung terkait proses eksperimen data berbasis A/B Testing secara end-to-end. Dalam program ini, peserta mempelajari bagaimana memahami business problem, melakukan validasi data, memilih metode statistical testing yang tepat, hingga menarik insight yang dapat digunakan untuk pengambilan keputusan bisnis.

Selama kegiatan berlangsung, peserta dikenalkan pada berbagai tahapan penting dalam A/B Testing, seperti pengecekan Sample Ratio Mismatch (SRM), Covariate Balance, dan Temporal Stability untuk memastikan data eksperimen valid sebelum dianalisis lebih lanjut. Selain itu, peserta juga mempelajari pemilihan statistical test berdasarkan jenis metric dan distribusi data, termasuk penggunaan t-test maupun Mann-Whitney test.

Studi kasus yang digunakan berasal dari industri e-commerce di Croatia dengan total lima eksperimen berbeda. Hasil eksperimen menunjukkan dampak yang beragam, mulai dari peningkatan revenue melalui personalized novelty slider dan product recommendation, hingga eksperimen yang memberikan dampak negatif seperti penggunaan dropdown menu.

Melalui program mentorship ini, peserta tidak hanya mempelajari teori A/B Testing, tetapi juga mendapatkan pengalaman praktik langsung dalam melakukan validasi eksperimen, analisis statistik, serta interpretasi hasil eksperimen untuk kebutuhan bisnis dan pengambilan keputusan berbasis data.

Hasil akhir analisis dan rekomendasi bisnis dituangkan dalam file report berjudul *“AB Testing Report_Ahmad Miftahul Farohi.pdf”*, sedangkan seluruh proses pengerjaan, data processing, dan kode program yang digunakan selama analisis tersedia pada folder *src*.
